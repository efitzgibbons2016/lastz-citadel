# lastz-citadel

This repository is for **documentation only**. It supports sharing and collaboration on LastZ-related materials to help alliance leaders run and grow their alliances.

## What’s in this repo

You’ll find written guides and reference materials for **LastZ** alliance management, including:

- **Alliance administration** — roles, permissions, and day-to-day operations  
- **Analysis** — how to read and use in-game data and reports  
- **Tactics & strategy** — approaches that work for competitive play and growth  

No code or technical setup is required. Everything here is meant to be read, shared, and used by leaders and admins in a business or gaming context.

## How to use this documentation

- **Browse online** — If this repo is hosted on GitHub, GitLab, or similar, you can open folders and files in your browser and read or download what you need.  
- **Download** — Use the platform’s “Download” or “Clone” option to get a copy of all documents on your computer. You don’t need to install any developer tools to open the files.  
- **Suggest changes** — If your host supports it, you can propose edits or new docs (e.g. via pull requests or issue templates). Otherwise, reach out to the repo owner to contribute or request updates.

## Contributing

If you have guides, templates, or strategies that would help other alliance leaders, we welcome contributions. Contact the repository owner or follow the contribution process described by the hosting platform to submit or suggest new documentation.
